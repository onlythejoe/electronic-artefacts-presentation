Infrastructure

Surface projects. Environment defines semantics.

The system is organized as a canonical core graph. Surface turns the system into a public projection, Environment stores the registry, Actions route changes, and extensions enrich the middle layer.

Surface → Environment → Actions → Extensions → Core

Core is the first canonical vertex. Inside it, only vertices and ties define topology.

Build

Studio designs experiences. Program codes the graph. Environment gives them type.

VASTE is built from capacities. Studio composes the public layer, Program orchestrates logical behavior, Environment defines the local registry for each vertex, extensions enrich the middle layer, and Assets lets any file type land on a vertex.

The core remains canonical; everything else attaches around it as composable capacity.
